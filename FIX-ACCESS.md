# Решение проблем с доступом к серверу

## Проблема: Сайт не открывается из сети

### Шаг 1: Проверьте, запущен ли сервер

Откройте командную строку и выполните:
```bash
netstat -an | findstr :8000
```

Должна быть строка: `TCP    0.0.0.0:8000           0.0.0.0:0              LISTENING`

Если сервера нет, запустите `start-server.bat`

### Шаг 2: Настройте файрвол (ОБЯЗАТЕЛЬНО!)

**Вариант А: Автоматически (требуются права администратора)**
1. Правой кнопкой мыши на `allow-port-8000.bat`
2. Выберите "Запуск от имени администратора"
3. Дождитесь сообщения "УСПЕШНО! Порт 8000 разрешен"

**Вариант Б: Вручную**
1. Нажмите `Win + R`, введите `wf.msc`, нажмите Enter
2. В левой панели выберите "Правила для входящих подключений"
3. В правой панели нажмите "Создать правило..."
4. Выберите "Порт" → "Далее"
5. Выберите "TCP" и "Определенные локальные порты"
6. Введите: `8000` → "Далее"
7. Выберите "Разрешить подключение" → "Далее"
8. Отметьте все три профиля (Домен, Частная, Публичная) → "Далее"
9. Введите имя: `Tushum Web Server` → "Готово"

### Шаг 3: Проверьте IP-адрес

Выполните в командной строке:
```bash
ipconfig | findstr /i "IPv4"
```

Используйте IP-адрес, который НЕ начинается с 127.0.0.1

### Шаг 4: Проверьте доступность

**На этом компьютере:**
- http://localhost:8000/login.html

**На других устройствах в сети:**
- http://[ВАШ_IP]:8000/login.html
- Например: http://172.20.10.10:8000/login.html

### Шаг 5: Если все еще не работает

1. **Проверьте, что устройства в одной сети**
   - Убедитесь, что оба устройства подключены к одной Wi-Fi сети

2. **Проверьте антивирус**
   - Временно отключите антивирус или добавьте исключение для порта 8000

3. **Проверьте роутер**
   - Убедитесь, что роутер не блокирует локальные соединения

4. **Попробуйте другой порт**
   - Измените порт в `start-server.bat` с 8000 на 8080
   - И настройте файрвол для нового порта

### Быстрая проверка

Выполните все команды по порядку:

```bash
# 1. Проверить сервер
netstat -an | findstr :8000

# 2. Проверить файрвол
netsh advfirewall firewall show rule name="Tushum Web Server"

# 3. Узнать IP
ipconfig | findstr /i "IPv4"

# 4. Проверить доступность (на другом устройстве)
ping [ВАШ_IP]
```

Если ping работает, но сайт не открывается - проблема в файрволе или порте.
